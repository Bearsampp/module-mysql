## Installation

* Download and install [Neard](https://github.com/crazy-max/neard).
* Download one of the available MySQL bundles.
* Extract archive in `neard\bin\mysql\`.

Directory structure :
```
[-] neard
 | [-] bin
 |  | [-] mysql
 |  |  | [-] mysql5.5.24
 |  |     | neard.conf
 |  |  | [-] mysql5.6.27
 |  |     | neard.conf
 ```
